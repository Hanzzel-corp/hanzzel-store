Hanzzel Store

100% estático · GitHub Pages Pagos externos seguros
Esta tienda funciona sin backend ni hosting pago: se despliega desde un repositorio en GitHub Pages. Los cobros se realizan mediante enlaces de pago (ej.: Mercado Pago, Stripe Payment Links, Lemon Squeezy, Gumroad). No se almacenan tarjetas en este sitio.

¿Cómo se procesa el pago?

Cada botón Comprar abre una URL de cobro externa (ej.: Mercado Pago en ARS o Stripe en USD). Puedes crear un enlace por producto y pegarlo en el catálogo (abajo).
